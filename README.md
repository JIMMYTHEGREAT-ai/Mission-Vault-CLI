# 🛡️ Mission Vault CLI
A professional-grade Command Line Interface (CLI) tool built in Python to track, categorize, and persist life missions.

## 🚀 Features
* **Persistence:** Uses JSON storage to save data even after the program exits.
* **Categorization:** Organize goals into Family, Health, Career, or custom categories.
* **Defensive Coding:** Includes error handling for missing data files.
* **Clean UI:** Professional formatting for high readability.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Data Format:** JSON (JavaScript Object Notation)

## 📖 How to Use
1. Run the script.
2. Select `1` to add a new mission.
3. Select `2` to view your encrypted vault.
4. Select `3` to exit securely.
